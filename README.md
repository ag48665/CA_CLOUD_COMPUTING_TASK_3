\# Cloud-Based Bus Pass System



\## Description



Cloud-Based Bus Pass System is a web application developed using Python Flask and SQLite.



The system allows users to:



\- Register an account

\- Log in securely

\- View available bus routes

\- Book bus tickets

\- Generate unique ticket numbers

\- View booking history

\- Cancel bookings



The application also contains an administrator panel for adding new bus routes.



\---



\## Technologies



\- Python 3

\- Flask

\- Flask-SQLAlchemy

\- SQLite

\- Flask-Bcrypt

\- HTML

\- CSS



\---



\## Installation



Create a virtual environment:



```bash

python -m venv venv

```



Activate it:



Windows



```bash

venv\\Scripts\\activate

```



Install packages



```bash

pip install -r requirements.txt

```



Run the application



```bash

python app.py

```
